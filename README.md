# it-universe-25-26
OSBB Management System — це веб-платформа для автоматизації адміністрування ОСББ (Об'єднання співвласників багатоквартирного будинку). Система дозволяє адміністраторам вести облік мешканців, контролювати фінансові операції, генерувати звітність та публікувати оголошення для мешканців. Проєкт на IT-Universe 25/26
