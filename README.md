# Freyja — Інформаційна система управління ОСББ для адміністраторів🏢

**Freyja** — це комплексна цифрова платформа, призначена для автоматизації процесів управління об'єднаннями співвласників багатоквартирних будинків (ОСББ). Проєкт розроблено з метою підвищення прозорості фінансових операцій, цифровізації документообігу та спрощення комунікації між адміністрацією будинку та його мешканцями.

Система орієнтована на забезпечення потреб невеликих та провінційних ОСББ, пропонуючи інтуїтивно зрозумілий інтерфейс та відсутність надлишкового, складного функціоналу.

🏆 _Проєкт представлено в рамках конкурсу IT-Universe 2026_

---

## 🏗 Архітектура проєкту

Система реалізована за моделлю B2B2C і складається з двох ключових взаємопов'язаних компонентів:

1. **Веб-панель адміністратора** (для керівництва ОСББ).
2. **Мобільний застосунок** (для мешканців багатоквартирного будинку).

---

## ⚙️ Основний функціонал

### 💻 Веб-платформа (Панель адміністратора)

- **Облік мешканців:** Створення, редагування та ведення бази даних власників квартир.
- **Фінансовий моніторинг:** Контроль заборгованостей, переплат та управління розрахунками платежів.
- **Генерація звітності:** Експорт фінансових звітів та відомостей по боржниках у форматі PDF.
- **Комунікаційний модуль:** Створення та публікація новин, оголошень і сповіщень для мешканців.
- **Адаптивність та UI:** Підтримка мобільних пристроїв та наявність темної теми оформлення.

---

## 🛠 Стек технологій

**Клієнтська частина (Веб):**

- `HTML5` / `CSS3` (забезпечення структури, стилізації та адаптивності).
- `JavaScript` (асинхронні AJAX-запити, клієнтська генерація PDF, валідація даних).

**Серверна частина (Веб):**

- `PHP` (обробка бізнес-логіки, маршрутизація, захист від SQL-ін'єкцій).
- `MySQL` (реляційна база даних для зберігання інформації про користувачів та фінанси).

---

## 🔒 Безпека та оптимізація

- Передача даних між клієнтом та сервером здійснюється за захищеним протоколом HTTPS.
- Паролі адміністраторів зберігаються у базі даних у хешованому вигляді.
- Реалізовано клієнтський підхід до формування PDF-документів для зниження навантаження на серверну інфраструктуру.
